HelloBlog 博客网
==================================

该项目是基于maven开发的其中

  前台使用          html+css+js
  后台使用          SpringMVC+Spring+Mybatis
  数据库使用的    mysql 5.7
  服务器使用       Tomcat 8.0
  JDK                  1.8/1.9
  开发工具           IDEA 2019

================================= Author: AiTboy

其他说明：

该项目开始于大三课程设计，没有什么技术含量。

主要功能
    用户的增删查改
    文章的增删查改
    评论的增删查改
    回复的增删查改
    最后加上一个搜索（luence）

该项目也是作者第一个javaweb项目，其中可能存在一些bug，并且很多共功能待完善。
